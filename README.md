## Local Installation

1. clone repo

```
git clone https://github.com/olarra/saagieTest
```

3. cd into directory

```
cd saagieTest
```

3. install node dependencies using npm or yarn

```
yarn OR npm i
```

3. Run Client

```
yarn run start-react
```

4. Run Server

```
yarn run start-api
```
